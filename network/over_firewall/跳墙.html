用谷歌服务器翻墙-GAppProxy代理安装使用详细教程

谷歌, 服务器, GAppProxy, 教程, 代理
前言（也叫废话）
本教程主要面向喜欢自己动手丰衣足食的朋友
有一定的动手能力和折腾的精神
本教程默认读者已经知道翻墙和代理的意思及电脑的一般操作
有一定翻墙经验者更适合本教程

一、基础知识准备


1、GAppProxy 是什么？
GAppProxy 是基于 Google app engine，为教育网用户提供一个免费的国际代理。由于借助了 Google 强大的服务器，所有也适用于公网的代理。

2、Google app engine 是什么？
Google app engine 是 Google 提供的一个在线应用程序平台，支持 Python。简单的说是在 Google app engine 上面直接运行用 Python 写的程序，由 Google app engine 提供网络空间和带宽。

3、用 GAppProxy 能干什么？
如果你在教育网，你可以把 GAppProxy 当作一个国际代理服务器，类似搜狗浏览器的教育网加速。
如果你在公网，正常情况下用不到 GAppProxy，但如果想访问某些特殊网站或者是被墙的网站，还是用得到的。
二、注册 Google App Engine:

1、用 Google 账户 登录 Google App Engine 后，点击 Create an Application 创建一个应用程序。

4、填写 Application Identifier (输入你想要的应用程序地址，相应会得到一个 yourname.appspot.com 的域名，记住这个) 和 Application Title (标题，随意啦)以及勾选同意服务条款，点 Save 即完成创建。

三、下载本次翻墙用到的软件

1、下载 Python 和 Google App Engine SKD  （如有更新以此行为准）
直接下载地址: Python | Google App Engine SDK  
（省事下载方式，如果突然无法访问，请将地址中的“http”换成“https”或者相反）
2、下载 GappProxy 和 fetchServer （同上）
直接下载地址: GappProxy | fetchServer  （同上）

四、安装软件并调整设置

1、Python 和 Google App Engine SKD 按照默认目录直接安装

2、解压缩 fetchServer 文件夹至 Google App Engine SKD 安装目录，使路径变为 "X:\Program Files\Google\google_appengine\fetchserver"
最终结果在 "X:\Program Files\Google\google_appengine\fetchserver" 文件夹中能找到 "app.yaml" 文件

3、用记事本打开刚解压的 app.yaml 文件，修改第一行 your_application_name 为刚才输入的 Application Identifier 就是yourname.appspot.com 中的 yourname。注意：紧接着"your_application_name"后面的"version"不要删除，否则会出错

4、上传 fetchserver

打开命令提示符(点击 开始 > 运行 > cmd)
进入 X:\Program Files\Google\google_appengine 目录(输入 cd X:\Program Files\Google\google_appengine\ 即可)，输入以下命令行: appcfg.py update fetchserver，回车，会要求你输入 Google 账户及密码，之后关闭结束上传。

5、测试 fetchserver：打开浏览器，进入 http://yourname.appspot.com/fetch.py 如果得到下面的页面，证明安装成功。

五、代理软件及浏览器设置

1、代理软件设置
将 GappProxy 解压至任意文件夹，打开 gui.exe，勾选 Use FetchServer
输入 http://yourname.appspot.com/fetch.py ，点 save，顺手点击 Status 确认代理运行正常。
