Thunderbird Extension: Folderpane Tools
Folderpane Tools 0.0.5.1
Released on 2007.04.11 for Thunderbird 1.0 - 3.0 on all operating systems
This extension allows for customization of the folder pane. Accounts can be rearranged and the startup folder can be chosen.

Translations available: en-US, de-DE, es-ES, fr-FR, it-IT, nl-NL, pl-PL, pt-BR, ru-RU, sk-SK
To install this extension, click on the Download button above and save the file in a convenient location. In Thunderbird, open the Tools menu and select Extensions. Click on the Install button and open the saved extension file.

Options
This extension must be configured before use. To open the options window, open the Tools menu, select Extensions, and double-click on the extension.
  • Override startup folder: Enable this option to override the folder that is displayed on startup.
    • Select folder: Select the folder to display at startup from the dropdown menu. To select a folder which has subfolders, open its submenu and select choose this folder.
  • Rearrange accounts: Use the listbox and buttons in this section to rearrange the order that accounts are displayed. The Default account is always listed first. Select an account from the listbox and click on one of the buttons below to either set the account as the default, move the account up in the list, or move the account down in the list.
    • Email accounts and RSS accounts are always placed above the Local Folders. The Local Folders are always placed above Newsgroup accounts. If the Local Folders or a Newsgroup account is set as the Default account, it will be placed at the top.

Bugs and Troubleshooting
  • WARNING: If you are using version 0.0.2 and see the message "58: fpAM.getAccount(accounts[i]).incomingServer has no properties" when you open the options window, please click Cancel for the window titled "Folderpane Tools Options". Otherwise, some accounts may be lost. To those who have been affected by this bug, please send me an email so I can help you recover your accounts.
  • If you see the message "Finishing Extension Installation..." after installing the extension, please read the instructions on how to fix it.
  • Currently, there is no option to select the root of an account as the startup folder. That feature will be implemented in a future version.
