In order to use Eduroam, users who run Ubuntu (or any other Linux distribution which uses NetworkManager) should start by telling NetworkManager to connect to the Eduroam network (in recent versions of Ubuntu this is accomplished by right-clicking on the networking icon in the top-right of the screen and then simply selecting the Eduroam network from the menu that appears).

NetworkManager should then display a configuration dialogue. The appropriate settings are as follows:

Security: WPA/WPA2 Enterprise
Authentication: PEAP
Anonymous Identity: leave blank
Inner Authentication: MSCHAP v2
PEAP Version: v0
Username: Your kent email address (e.g. xyz123@kent.ac.uk - if you are a member of staff, you must use your short-form address)
Password: Your Kent network password

This configuration was tested on Ubuntu Intrepid (8.10). Older distributions may not include a WPA-enabled NetworkManager; you should refer to your distribution's community support for instructions on how to enable the above configuration.
